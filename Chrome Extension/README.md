# Leads Tracker Chrome Extension

一个简单实用的Chrome扩展，帮助你快速保存和管理网页链接。

## 功能特性

- **一键保存** - 点击按钮即可保存当前标签页URL
- **手动输入** - 支持手动添加任意链接
- **本地存储** - 数据保存在浏览器本地，隐私安全
- **快速访问** - 点击链接直接在新标签页中打开
- **批量删除** - 双击删除按钮清空所有保存的链接

## 安装使用

1. 下载项目文件
2. 打开Chrome扩展管理页面 (`chrome://extensions/`)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
